# IoT Data Dashboard for Smart Home Management

## Overview
The IoT Data Dashboard for Smart Home Management is a sophisticated platform designed to simplify the oversight of smart home devices and user interactions. This project offers a centralized interface for monitoring and controlling a variety of IoT devices, making it an essential tool for homeowners and administrators aiming to efficiently manage device statuses, user activities, and event logs. By combining an intuitive dashboard with powerful backend services, the project tackles the complexities of managing multiple smart devices within a home environment. It is particularly valuable for smart home enthusiasts, IoT developers, and system administrators who require a dependable solution for device operations and user management.

The dashboard provides real-time insights into device statuses and events, enabling users to swiftly address any issues or alerts. With a secure user management system, it ensures that only authorized individuals can access and control the smart home devices. The project is developed using FastAPI for the backend and SQLite for data storage, ensuring a lightweight and efficient solution.

## Features
- **Device Management**: Easily add, view, and manage smart home devices.
- **User Management**: Create and manage user accounts with roles and permissions.
- **Event Monitoring**: Track and review events and alerts generated by devices.
- **Responsive Dashboard**: Access the dashboard from any device with a modern, responsive design.
- **Secure Access**: Authentication mechanisms ensure secure access to the dashboard and APIs.
- **Real-time Updates**: Receive instant updates on device status and events.
- **Template-based UI**: Utilizes Jinja2 for dynamic HTML rendering.

## Architecture
The project is structured to separate concerns between the frontend and backend. The backend, built with FastAPI, serves API endpoints and HTML pages using Jinja2 templates. The SQLite database stores data for devices, users, and events.

```plaintext
+-----------------+
|  Frontend       |
|  (HTML/CSS)     |
+--------+--------+
         |
         |
+--------v--------+
|  FastAPI        |
|  (Backend)      |
+--------+--------+
         |
         |
+--------v--------+
|  SQLite         |
|  (Database)     |
+-----------------+
```

### Data Flow
1. User accesses the dashboard through a web browser.
2. FastAPI serves the requested HTML page.
3. User actions trigger API requests to FastAPI endpoints.
4. FastAPI interacts with the SQLite database to fetch or update data.
5. Responses are sent back to the frontend for display.

## Getting Started

### Prerequisites
- Python 3.11+
- pip (Python package manager)
- Docker (optional, for containerization)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/iot-data-dashboard-for-smart-home-management-auto.git
   cd iot-data-dashboard-for-smart-home-management-auto
   ```
2. Install the required Python packages:
   ```bash
   pip install -r requirements.txt
   ```

### Running the Application
1. Start the FastAPI application using Uvicorn:
   ```bash
   uvicorn app:app --reload
   ```
2. Open your web browser and visit `http://127.0.0.1:8000` to access the dashboard.

## API Endpoints
| Method | Path             | Description                        |
|--------|------------------|------------------------------------|
| GET    | /api/devices     | Retrieve a list of all devices     |
| POST   | /api/devices     | Add a new device                   |
| GET    | /api/events      | Retrieve a list of all events      |
| POST   | /api/users       | Create a new user                  |
| GET    | /api/users       | Retrieve a list of all users       |
| GET    | /                | Render the main dashboard page     |
| GET    | /devices         | Render the devices management page |
| GET    | /events          | Render the events page             |
| GET    | /users           | Render the user management page    |
| GET    | /settings        | Render the settings page           |
